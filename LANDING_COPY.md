3. **What is Credex?**
   Credex helps startups buy discounted AI infrastructure credits. If our audit finds high savings, we can help you capture them by sourcing the same tools through our secondary credit marketplace.
4. **Does this work for API spend too?**
   Yes. We analyze both seat-based subscriptions and usage-based API spend (OpenAI, Anthropic, Gemini).
